基本的なHooks

  • useGetDoc（ドキュメントの取得）
  • useGetCollection（コレクションの取得）
  • useSubscribeDoc（ドキュメントの監視）
  • useSubscribeCollection（コレクションの監視）
  • useQuery（クエリオプションを用いたドキュメント・コレクションの取得・監視）

高度なHooks

  • useLazyGetDoc（ドキュメントの遅延取得）
  • useLazyGetCollection（コレクションの遅延取得）
  • usePaginateCollection（ページネーションを用いたコレクション取得）

概要

Fireclient は Firestore を用いたモダンな React アプリケーションにおけるネットワークレイヤーとステート管理レイヤーを担うライブラリです。Fireclient は Firestore をバックエンドとして持つ複雑な React アプリケーションのデータ取得における一つの解を提供します。
